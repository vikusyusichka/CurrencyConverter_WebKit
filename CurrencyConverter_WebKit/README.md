# Simple Currency Converter

A simple iOS app demonstrating **WebKit (WKWebView)** integration in SwiftUI. The app loads a currency conversion website and provides basic navigation controls.

## Features

- Load an external currency conversion website  
- Show a loading indicator while the page loads  
- Back, Forward, and Reload navigation  
- SwiftUI interface with WKWebView wrapped in an ObservableObject  

## Tech

- Swift / SwiftUI  
- WebKit (WKWebView)  
- Combine for reactive state
